# Crop-Recommendation-system-using-Machine-Learning



To create a Flask application for crop recommendation based on climate and soil strength, start by installing Flask using a requirements.txt file. Set up the Flask app with routes for the home page and a recommendation page to handle user input. Implement a function to provide crop suggestions based on the provided climate and soil strength. Create HTML templates for the input form and for displaying the recommendations. Finally, run the application and navigate to the provided URL to access and interact with the app.
